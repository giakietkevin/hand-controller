# hand-controller
Ý tưởng: Hệ thống điểm danh/điều khiển bằng cử chỉ tay (Virtual Mouse/Hand Controller)  Mô tả: Người dùng đưa tay trước camera để điều khiển chuột, slide thuyết trình hoặc chơi game đơn giản (như Flappy Bird).  Thư viện: MediaPipe Hands (rất nhẹ, chạy mượt trên trình duyệt).  Ưu điểm: Không cần server xử lý ảnh, deploy cực nhanh lên GitHub Pages.
